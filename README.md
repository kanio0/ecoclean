Eco Clean Product 


## Что внутри
- `frontend/` — React + Vite SPA
- `backend/` — Express API с mock-данными

## Страницы
- Главная
- Карта точек
- Разведка / добавить точку
- Награды
- Команда
- Рейтинг
- Профиль

## Запуск
### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Адреса
- Frontend: http://localhost:5173
- Backend: http://localhost:4000
